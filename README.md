# 🍺 Brew Addon
**"A Meteor addon for the drunks."**  

## 🧪 Installation

1. Make sure you have [Meteor Client](https://meteorclient.com) installed.
2. Drop `brew-addon.jar` into your `.minecraft/mods` folder.
